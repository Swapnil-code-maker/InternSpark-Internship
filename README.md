# 🔹 Task 1 — Student Management REST API

## Features

- Create Student
- Get All Students
- Get Student by ID
- Update Student
- Delete Student

## Technologies Used

- Spring Boot
- Spring Web
- Spring Data JPA
- MySQL
- Postman

## API Endpoints

| Method | Endpoint | Description |
|---|---|---|
| POST | `/students` | Add Student |
| GET | `/students` | Get All Students |
| GET | `/students/{id}` | Get Student by ID |
| PUT | `/students/{id}` | Update Student |
| DELETE | `/students/{id}` | Delete Student |

# ▶️ How to Run the Projects

## 1. Clone Repository

```bash
git clone https://github.com/Swapnil-code-maker/InternSpark-Internship.git
```

---

## 2. Open Project in IntelliJ IDEA

Open the required task folder:
- Task-1-Student-Management-API
- Task-2-Database-Migration-API

---

## 3. Configure MySQL Database

Update:
- database name
- username
- password

inside:

```text
application.properties
```

---

## 4. Run Spring Boot Application

Run:

```text
Application.java
```

---

## 5. Test APIs Using Postman

Use Postman to test all REST API endpoints.
