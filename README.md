# ✅ Completed Tasks

## 🔹 Task 1 — Student Management REST API

### Features

* Create Student
* Get All Students
* Get Student by ID
* Update Student
* Delete Student

### Technologies Used

* Spring Boot
* Spring Web
* Spring Data JPA
* MySQL
* Postman

### API Endpoints

| Method | Endpoint         | Description       |
| ------ | ---------------- | ----------------- |
| POST   | `/students`      | Add Student       |
| GET    | `/students`      | Get All Students  |
| GET    | `/students/{id}` | Get Student by ID |
| PUT    | `/students/{id}` | Update Student    |
| DELETE | `/students/{id}` | Delete Student    |
