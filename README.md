# 🚀 InternSpark Java Internship

This repository contains all internship tasks, Java backend projects, and software engineering practice completed during the **InternSpark AICTE-listed Internship Program**.

The projects focus on backend development using Java, Spring Boot, MySQL, REST APIs, database integration, unit testing, and logging concepts.

---

# 👨‍💻 Tech Stack

- Java 17
- Spring Boot
- Spring Data JPA
- MySQL
- REST API Development
- Flyway Migration
- JUnit 5
- Mockito
- SLF4J / Logback
- Maven
- Postman
- Git & GitHub
- Object-Oriented Programming (OOP)

---

# 📌 Internship Goals

- Improve Java backend development skills
- Build real-world REST APIs
- Practice clean code architecture
- Learn database integration and migrations
- Understand unit testing and logging
- Understand Git & GitHub workflow
- Strengthen debugging and problem-solving ability
